# VoiceDraft Studio 実装仕様書

この文書は、VoiceDraft Studio の実装方針と構成をまとめたものです。

## 1. 目的

口述筆記から、ブログ記事やクラウドファンディング本文を生成する支援アプリを提供する。

## 2. システム構成

- フロントエンド: HTML + TypeScript
- バックエンド: FastAPI
- データストア: Firestore
- 画像保存: Cloud Storage
- デプロイ: Firebase Hosting / Cloud Run

## 3. MVP機能

- 口述入力
- 下書き保存
- 要点抽出
- ブログ生成
- クラウドファンディング本文生成
- 画像生成プロンプト案生成

## 4. データモデル

### drafts

- draft_id
- tenant_id
- author
- raw_text
- extracted_points
- status
- created_at
- updated_at

### generated_texts

- text_id
- draft_id
- output_type
- title
- body
- prompt_used
- model
- created_at

### generated_images

- image_id
- draft_id
- text_id
- image_type
- prompt
- image_url
- created_at

## 5. API

- POST /drafts
- GET /drafts
- GET /drafts/{draft_id}
- POST /drafts/{draft_id}/extract
- POST /generate/blog
- POST /generate/crowdfunding
- POST /generate/image/prompt-draft
- POST /generate/image
