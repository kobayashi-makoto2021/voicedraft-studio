# VoiceDraft Studio 費用見積もり

## 1. 前提

MVPでは、以下の構成を想定する。

- フロントエンド: Firebase Hosting
- バックエンド: Cloud Run
- データ保存: Firestore
- 画像保存: Cloud Storage
- AI利用: Claude API / OpenAI API

## 2. 主要な費用項目

### 2.1 Claude API

想定用途:
- 要点抽出
- ブログ本文生成
- クラファン本文生成
- 画像生成プロンプト案生成

目安:
- 1回の要点抽出: 1〜3 リクエスト
- 1回の本文生成: 1 リクエスト
- 1日あたりの利用ユーザーが 10名、1名あたり 3〜5回利用した場合を想定

概算:
- 月間 50〜100 リクエスト程度なら、数百円〜1,000円台の可能性が高い
- 利用量が増えると従量課金のため増加する

### 2.2 OpenAI API（DALL-E）

想定用途:
- 画像生成

目安:
- 1枚生成あたり 1 リクエスト
- 1ユーザーあたり 1〜3枚生成を想定

概算:
- 1枚あたりのコストは画像サイズやモデルにより変動
- 100枚/月程度なら、比較的低コストだが、利用が増えると増加

### 2.3 Firebase Hosting

想定用途:
- フロントエンド配信

目安:
- 静的配信のため、通常は低コスト
- ほぼ無料枠内に収まることが多い

### 2.4 Cloud Run

想定用途:
- FastAPI バックエンドの実行

目安:
- 1インスタンスの常時稼働はコストがかかるため、必要時のみ起動する構成が望ましい
- MVP段階では、使用量が少なければ低コスト寄り

### 2.5 Firestore

想定用途:
- drafts
- generated_texts
- generated_images
- tenant_settings

目安:
- 小規模利用なら月額数百円〜数千円程度で収まることが多い
- 読み書き回数が増えるほどコスト増

### 2.6 Cloud Storage

想定用途:
- 生成画像の保存

目安:
- 画像数が少なければ低コスト
- 生成画像を数十〜数百枚保存するだけなら大きな負担にはなりにくい

## 3. 目安の月額費用

| 項目 | 目安 |
|---|---:|
| Claude API | 数百円〜数千円 |
| OpenAI API | 数百円〜数千円 |
| Firebase Hosting | ほぼ無料〜数百円 |
| Cloud Run | 数百円〜数千円 |
| Firestore | 数百円〜数千円 |
| Cloud Storage | 数百円以下 |

## 4. 使い方ごとの想定

### 4.1 個人利用・試作段階

- 利用頻度が低い
- 月数十リクエスト程度
- ほぼ無料に近い範囲

### 4.2 小規模業務利用

- 1日数回〜数十回の利用
- 生成画像も少数
- 月額 2,000円〜1万円程度を見込む

### 4.3 本格運用

- 複数ユーザー・高頻度利用
- 画像生成も多い
- 月額 1万円以上になる可能性あり

## 5. コスト削減のポイント

- APIキーは必要な機能だけに限定する
- 画像生成は必要時のみ実行する
- Cloud Runは必要時のみスケールさせる
- 生成画像は圧縮や期限付き保持を検討する
- ローカル開発時はモック/ダミーデータを活用する

## 6. 追加で確認したい項目

- Claude APIの実際の利用量
- OpenAI APIの画像生成枚数
- Firestoreの読み書き回数
- Cloud Runの稼働時間
